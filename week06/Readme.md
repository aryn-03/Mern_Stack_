# Week 06 Projects

A collection of 6 modern frontend projects demonstrating HTML and CSS layouts, responsive design patterns, and web UI components.

## Projects Overview

### 1. 📄 Landing Page
**Folder:** `landing-page/`

A marketing-focused landing page template featuring a prominent hero section (jumbotron) with call-to-action button and three feature cards showcasing key benefits. Perfect for startups or service-based businesses.

**Components:** Hero section, Feature cards, Call-to-action buttons

---

### 2. 🏛️ Libertarian Manifesto
**Folder:** `libertarian-website/`

A political advocacy website with bold red theme and impactful messaging. Features a hero section with flame icon, top navigation bar, and informational cards discussing libertarian principles and ideology.

**Components:** Top bar navigation, Hero section, Content cards, Semantic messaging

---

### 3. ✈️ Adventure Tours
**Folder:** `adventure-tour/`

A travel and tourism platform showcasing outdoor activities and expedition services. Includes hero banner, outdoor activity section with image slider, tourism type grid (hiking, rafting, winter expeditions), and call-to-action elements.

**Components:** Hero with overlay, Image sliders, Activity grid, Tourism icons, Responsive layout

---

### 4. 💬 Testimonials Grid
**Folder:** `grid/`

A responsive grid layout displaying user testimonials and reviews. Features profile cards with avatars, verified graduate badges, and testimonial quotes. Great for showcasing customer success stories.

**Components:** Card grid layout, Profile sections, Avatar images, Testimonial cards

---

### 5. 🛍️ Product Grid Layout
**Folder:** `responsive-color/`

A modern product showcase and marketing grid layout. Displays featured products (iPhone 6S), educational content, and promotional items in an organized, visually appealing grid format.

**Components:** Product cards, Grid layout, Marketing copy, Image integrations

---

### 6. 🏙️ City Guide
**Folder:** `tailwind/`

A city guide and dining directory website with navigation header featuring food categories (ice cream, donuts, tea, coffee). Includes hero section and curated listings of local establishments.

**Components:** Navigation bar, Hero section, Category links, Responsive design

---

## Project Structure

Each project folder contains:
- `index.html` - Main HTML file with semantic markup
- `styles.css` - Complete styling and responsive design rules

## Getting Started

1. Navigate to any project folder
2. Open `index.html` in your web browser
3. All styling is self-contained in `styles.css`

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Responsive Design, Media Queries
- **External Assets** - Images from Unsplash, Icons from Flaticon
